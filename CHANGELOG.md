# Changelog

All notable changes to this project will be documented in this file.

## [0.1.1] - 2026-01-11

### Added
- Standard JSON Pointer escaping for diff/merge paths, plus conflict context fields for merge previews.
- OpenAPI contract checks/linting scripts and CI enforcement.
- Release documentation and changelog tracking.

### Changed
- Conflicts now include unit summary, refContext, and optional pathTokens for easier UI handling.
- OpenAPI/SDK updates to reflect the enriched conflict payloads.
